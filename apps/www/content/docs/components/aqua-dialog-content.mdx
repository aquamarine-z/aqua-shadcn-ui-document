## Installation

<CodeTabs>

  <TabsList>
    <TabsTrigger value="cli">CLI</TabsTrigger>
  </TabsList>
  <TabsContent value="cli">
    ```bash
    npx shadcn@latest add https://aqua-shadcn-ui.vercel.app/r/aqua-dialog-content.json
    ```
  </TabsContent>

## Usage

```typescript jsx
return <Dialog>
<AquaDialogContent>
    <DialogTitle>
        Hello World
    </DialogTitle>
</AquaDialogContent>
</Dialog>
```

## Examples
### Hide Close Button

```typescript jsx
return <Dialog>
<AquaDialogContent closeButton={false}>
    <DialogTitle>
        Hello World
    </DialogTitle>
</AquaDialogContent>
</Dialog>
```
